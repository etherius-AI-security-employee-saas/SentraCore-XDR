# SentraCore XDR Manager And Employee Guide

## Current release status

The current SentraCore XDR release is a **manager / security console build**.

It is strong for:

- local setup
- dashboard testing
- attack simulations
- product demos

It does **not yet** include:

- real employee endpoint agent
- automatic employee enrollment
- direct employee-to-manager auto-connect

## What the manager should download

Download:

- `Setup SentraCore XDR.cmd`

This is the correct installer for the current release.

It installs:

- GUI dashboard
- backend API
- AI engine

## What the employee should download

Right now:

- there is no separate real employee installer in this version
- employees should not be treated as production-managed agents yet

## If you want the full manager-to-employee workflow

The next version must add:

- manager login
- company creation
- employee enrollment key
- employee background agent
- secure backend registration
- manager dashboard tied to enrolled employees
